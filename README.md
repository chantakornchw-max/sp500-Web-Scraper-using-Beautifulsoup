## Project Overview
This project demonstrates a Python-based web scraping pipeline designed to extract real-time financial data for the S&P 500 companies. By targeting a leading financial analysis platform, the script automates the collection of key performance metrics, transforming raw HTML into a structured format for further analysis.

## Technical Stack & Libraries
• Python: Core programming language.

• requests: Handles HTTP requests to retrieve web content.

• BeautifulSoup (bs4): Parses HTML and extracts specific data elements.

• pandas: Structures the scraped data into a clean DataFrame and exports to CSV.

## Dataset & Features
The scraper successfully extracts the following features for all 500+ constituent companies:

• Symbol & Company Name: Unique identifiers for each stock.

• Market Cap: Total market value (represented in Trillions/Billions).

• Stock Price: Current trading price per share.

• % Change: Daily price volatility percentage.

• Revenue: Total income generated by the company.

## Data Snapshot & Timing
• Scrape Date: Saturday, March 14, 2026.

• Data Reference: This dataset reflects the Market Close prices from Friday, March 13, 2026 (4:00 PM ET), as the US stock markets are closed on weekends.
